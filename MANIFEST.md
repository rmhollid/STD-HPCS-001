- AlN / ScAlN
- Al2O3 / sapphire / alumina
- diamond / DLC
- YSZ / yttria-stabilized zirconia
- HfO2 / ZrO2 refractory oxides
- MgO / MgAl2O4 spinel
- rare-earth-doped YAG
- rare-earth-doped Y2O3 / Lu2O3 / Sc2O3 transparent ceramics

- phase-change spectral materials
